# 超级浏览器

一款基于 Electron 构建的功能完整的桌面浏览器，支持视频嗅探下载和多语言翻译。

## 功能特性

### 核心浏览功能
- **多标签页浏览** - 支持同时打开多个网页标签
- **地址栏搜索** - 支持网址直达和搜索引擎搜索
- **前进/后退/刷新** - 完整的导航控制
- **主页设置** - 自定义浏览器主页

### 视频嗅探下载
- **自动媒体检测** - 自动嗅探网页中的视频和音频资源
- **多种格式支持** - 支持 MP4、WebM、M3U8、MP3、WAV 等格式
- **一键下载** - 点击即可下载嗅探到的媒体文件
- **下载管理** - 查看下载进度和已完成的下载

### 翻译功能
- **文本翻译** - 支持多语言文本翻译（中文、英语、日语、韩语、法语、德语、西班牙语、俄语）
- **页面翻译** - 一键翻译整个网页内容
- **自动语言检测** - 自动识别源语言

### 书签与历史
- **书签管理** - 添加、删除、访问书签
- **历史记录** - 自动记录浏览历史，支持清除

### 个性化设置
- **深色模式** - 支持深色/浅色主题切换
- **搜索引擎选择** - 百度、Google、必应
- **下载目录设置** - 自定义文件下载位置
- **数据持久化** - 书签、历史、设置自动保存

## 安装说明

### 环境要求
- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 克隆或下载项目
```bash
cd electron-browser
```

2. 安装依赖
```bash
npm install
```

3. 运行开发模式
```bash
npm run dev
```

4. 打包为可执行文件

**Windows:**
```bash
npm run build:win
```

**macOS:**
```bash
npm run build:mac
```

**Linux:**
```bash
npm run build:linux
```

### 使用打包脚本

**Windows (双击运行):**
```
build.bat
```

**Linux/macOS:**
```bash
chmod +x build.sh
./build.sh
```

## 使用说明

### 基本操作
- **新建标签页** - 点击标签栏右侧的 + 按钮
- **关闭标签页** - 点击标签页上的 × 按钮
- **切换标签页** - 点击标签页即可切换
- **地址栏** - 输入网址或搜索关键词，按回车访问

### 视频下载
1. 打开包含视频的网页
2. 点击工具栏上的 "媒体嗅探" 按钮
3. 在媒体面板中查看检测到的视频
4. 点击下载按钮即可下载

### 翻译功能
1. **文本翻译**:
   - 点击工具栏上的 "翻译" 按钮
   - 在翻译面板中输入要翻译的文本
   - 选择源语言和目标语言
   - 点击 "翻译" 按钮

2. **页面翻译**:
   - 打开需要翻译的网页
   - 点击 "翻译" 按钮打开翻译面板
   - 选择目标语言
   - 点击 "翻译当前页面" 按钮

### 书签管理
- **添加书签** - 点击地址栏右侧的书签图标
- **查看书签** - 点击工具栏上的书签按钮
- **删除书签** - 在书签面板中点击删除按钮

## 注意事项

1. **图标文件** - 打包前请确保 `assets` 目录中包含所需的图标文件
2. **网络环境** - 翻译功能需要访问 Google 翻译 API，请确保网络畅通
3. **下载目录** - 首次使用请设置下载目录，默认为系统下载文件夹
